# Mussel-Classification

# Philippine Mussel Classification using MobileNetV3 (Edge AI)

## Overview
This project is a real-time image classification system designed to identify three Philippine mussel species using a Raspberry Pi 4.

## Features
- Real-time image capture using Pi Camera
- MobileNetV3-based image classification
- TensorFlow Lite inference for edge deployment
- TFT display output for predictions

## Technologies Used
- Python
- TensorFlow Lite
- OpenCV
- Raspberry Pi 4

## Results
- Achieved 95% classification accuracy across 3 classes
- Optimized for low-power, real-time inference

## System Setup
1. Capture image using Pi Camera
2. Preprocess input
3. Run inference using TFLite model
4. Display classification result
